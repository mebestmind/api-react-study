## 메타/관리 제안

- 각 문서 상단에 다음 메타 정보를 추가하면 문서 관리와 자동화에 유리합니다 (예: GitHub Pages, MkDocs 등):

```yaml
---
title: "문서 제목"
summary: "한 줄 요약"
version: "v1.0"
last_updated: "2026-07-10"
---
```

- 원하시면 제가 각 문서에 위 front-matter(메타) 블록을 자동으로 추가해 드리겠습니다.
- 문서가 더 늘어나면 `docs/README.md`를 사이드바용 인덱스(예: MkDocs `nav`) 형식으로 변환해 드릴 수 있습니다.
