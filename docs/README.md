# 문서 색인 (docs)

이 폴더에는 프로젝트 관리·설명 문서를 모아두었습니다. 아래 목록에서 필요한 문서를 클릭해 상세 내용을 확인하세요.

- [관리 및 배포 가이드](./manage.md) — 로컬에서 프로젝트 생성하는 방법, React/Python 환경 설정, Git 업로드(커밋·푸시) 절차를 정리한 실무 가이드입니다.
- [요약: 초기 버전](./Summary.md) — 프로젝트 아키텍처와 초기 설정(프론트엔드/백엔드) 가이드의 요약본입니다. 초기에 따라하기 좋은 단계별 설명을 포함합니다.
- [요약: 업그레이드 버전](./Summary2.md) — 디렉토리 구조, 파일별 역할, API 사양 등 확장된 사양 문서입니다. 배포 전 확인용 체크리스트와 보안 고려사항을 포함합니다.

추가 제안

- 문서가 늘어나면 `docs/README.md`를 목차(TOC)로 유지하고, 각 문서에 `front-matter`(제목, 요약)를 추가하면 자동 문서 생성(예: GitHub Pages) 또는 탐색에 유리합니다.
- 원하시면 각 문서에 요약(한 줄 설명)을 추가하거나, 문서 간의 링크를 강화(예: Summary → manage)해 드리겠습니다.
